Solution: Deity Duos
Answer: FATES

Written by Skynet000

The flavortext references the game Hades, which has a mechanic called duo boons. For a duo boon, two of the Olympian gods in the game collaborate to produce some power-up for Zagreus. There are 28 such boons, one for each possible pair of the 8 gods.

Each duo boon has a unique bigram consisting of the first letter of each word (ignoring the 3 “Curse of …”, which have unique bigrams if you remove “of” anyway). This allows solvers to determine which bigrams correspond to which boons.

Then, the goal of this puzzle is to arrange the given boons so that adjacent boons share a god. A more visualizable way of describing this: imagine that each god represents a different color, and you are given discs that were colored half one color and half another. Your goal would be to arrange the discs in a circle so that all touching edges were the same color.

After finding the unique arrangement with the given discs in their fixed positions, index into the full duo boon’s name with the number at that position to extract a letter. Reading clockwise from the indicated starting circle yields the cluephrase THE MOIRAI (ENGLISH), which fits the given enumeration at the bottom. The answer to this cluephrase, and thus the puzzle, is the 5 letter word for Moirai in English, or FATES.
